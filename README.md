# my-website

This is a minimal Vite + React + Tailwind project containing a single-file portfolio app (`src/App.jsx`) provided by the user.

Quick start (PowerShell):

```powershell
cd my-website
npm install
npm run dev
```

Then open the local dev URL shown by Vite (usually `http://localhost:5173`).

Build for production:

```powershell
npm run build
npm run preview
```
